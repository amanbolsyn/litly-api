# Litly API



## Installation

Clone the repository 
```bash
git clone https://github.com/amanbolsyn/litly-api.git
```

Go to the cloned directory
```bash
cd litly-api/
```

Build docker containers
```bash
make install
```

Start docker containers
```bash
make run
```

Stop docker containers
```bash
make down
```

Run new db migrations 
```bash
make migrate
```

Seed the database
```
make seed
```

 Run already migreated migrations
```
make fresh
```
